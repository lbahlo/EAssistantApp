What is NAEP Assistant

  • A 'Chatbot' that is designed to help with NAPE Assessment 'help desk' issues.

How to use NAEP Assistant

  • It is very simple.
  • You ask questions by clicking on the microphone icon and speaking. (Remember: when it goes purple, NAPE Assistant is listening.)
  • Or by typing your question in the 'Your Message' input bar, typing enter or pressing the send button.
  • And then NAEP Assistant will start a conversation with you to help solve the problem. Give it a try, it's fun.

Current "Topics" NAEP Assistant has knowlege on

  • Router Trouble
  • No Audio (coming soon)
  • Internet (coming soon)

Here is an example of a conversation you can have with NAPE Assistant

  • user => "Talk with NAEP Assistant">
  • NAEP => "NAEP Assistant here. How can I help?"
  • user => "I am having trouble with my Router.">
  • NAEP => "What is the problem you are having with the Router?"
  • user => "I see no lights.">
  • NAEP => "The router is not displaying any lights, correct?"
  • user => "That is correct.">
  • NAEP => "Solution 1: Make sure the power cord is plugged securely into the router and the outlet or power strip. Solution 2: Try a different outlet. Ensure power strip is turned on and working. Solution 3: Try the power cord from your backup router."
  • user => "That Worked."
  • NAEP => "That is great to hear. Can I help you with anything else?">
  • user => "No not right now. Bye."
  • NAEP => "Ok. Glad to be of help. Bye.>
